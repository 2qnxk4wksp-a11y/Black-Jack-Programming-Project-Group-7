# Group-Programming-Project-Black-Jack
Procedural Programming COSC-1310-010

Brennen Burks
Evan Strevey

This is a Blackjack card game programmed in C23.
You start with 100$ and lose when you run out of money.
Features:
Betting system,
Hit or Stand feature,
Real card functions and random drawing,
Time between turns,
Anti-crash from incorrect input,
Million-dollar win limit.
